# Mandoline Design System (macOS SwiftUI)

## Visual Direction
A flat, minimalist design inspired by macOS Sequoia. The aesthetic focuses on clarity, purposeful whitespace, and a high-quality "silverware" feel through refined typography and a consistent, professional color palette.

## Typography
- **Logo Font:** Merriweather (Serif) - used for branding to convey quality and precision.
- **System Font:** San Francisco (Sans-Serif) - used for all UI elements, labels, and body text to ensure native macOS feel.

## Color Palette

### Core Colors
- **Primary Action:** `#7AB4E1` (Soft Sky Blue) - Used for primary buttons and active states.
- **Background (Main):** `#F9F9F9` (Off-White) - A clean, light background for main content areas.
- **Background (Subtle):** `#E8F1F8` (Pale Blue-Grey) - Used for secondary backgrounds or decorative elements.
- **Text (Primary):** `#2D2D2D` (Deep Charcoal) - For high-readability titles and body text.
- **Text (Secondary):** `#6E6E6E` (Medium Grey) - For labels, hints, and less prominent information.

### UI Element Colors
- **Button Surface:** `#E0E0E0` (Light Grey) - For secondary actions.
- **Border / Divider:** `#D1D1D1` (Soft Grey) - For clean separation of sections.
- **Success:** `#4CAF50` (Standard Flat Green)
- **Error/Danger:** `#E57373` (Soft Flat Red)

## Component Guidelines
- **Corners:** Use a standard corner radius of `12pt` for main containers and `8pt` for buttons to match modern macOS standards.
- **Shadows:** Avoid drop shadows. Use subtle borders (`1px`) for depth if necessary, or lean on color contrast.
- **Icons:** Use flat, 2D vector icons with consistent line weights.
